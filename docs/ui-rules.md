# قواعد الواجهة والتصميم

## اللغة والاتجاه

- الواجهة عربية بالكامل.
- الاتجاه RTL.
- استخدام عبارات قصيرة وواضحة.
- تجنب الازدحام في شاشة الجوال.

## الشاشة الرئيسية

العنوان المقترح:

> صمّم تهنئة العيد باسمك خلال ثواني 🎁

النص المساعد:

> اختر التصميم، اكتب الاسم، وحمّل بطاقتك الجاهزة للمشاركة.

## أزرار رئيسية

- معاينة البطاقة
- تحميل PNG
- مشاركة
- نسخ نص التهنئة

## مقاس البطاقة

المقاس الأساسي:

- 1080 × 1920
- مناسب للواتساب وإنستقرام والمنشورات العامة.

## القوالب المبدئية

### قالب 1: فاخر ذهبي

- خلفية داكنة أو متدرجة.
- زخارف هلال ونجوم.
- نص ذهبي أو أبيض.
- مناسب للتهاني الرسمية والعائلية.

### قالب 2: ناعم وبسيط

- ألوان فاتحة.
- عناصر زخرفية خفيفة.
- مناسب للأمهات والأقارب.

### قالب 3: شبابي مرح

- ألوان زاهية.
- عناصر عيد بسيطة.
- مناسب للأصدقاء والمتابعين.
