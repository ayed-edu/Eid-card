# عقود البيانات

## نموذج بيانات البطاقة

```json
{
  "recipientName": "عمي الغالي محمد",
  "greetingText": "كل عام وأنت بخير، وتقبّل الله منا ومنكم صالح الأعمال.",
  "senderName": "محبك أبو محمد",
  "templateId": "gold-luxury",
  "exportFormat": "png"
}
```

## نموذج القالب

```json
{
  "id": "gold-luxury",
  "name": "فاخر ذهبي",
  "description": "تصميم داكن بزخارف ذهبية مناسب للتهاني الرسمية.",
  "ratio": "1:1",
  "width": 1080,
  "height": 1920,
  "defaultTexts": {
    "recipientName": "عمي الغالي محمد",
    "greetingText": "كل عام وأنت بخير، وتقبّل الله منا ومنكم صالح الأعمال.",
    "senderName": "محبك أبو محمد"
  }
}
```

## نص المشاركة

يبنى النص تلقائيًا من اسم المهدى إليه:

```text
إهداء إلى {recipientName} 🌙
كل عام وأنت بخير، وتقبّل الله منا ومنكم صالح الأعمال.
```

مثال:

```text
إهداء إلى عمي الغالي محمد 🌙
كل عام وأنت بخير، وتقبّل الله منا ومنكم صالح الأعمال.
```

## قواعد التحقق من الإدخال

- إذا ترك المستخدم اسم المهدى إليه فارغًا، يستخدم النص الافتراضي.
- الحد المقترح لاسم المهدى إليه: 40 حرفًا.
- الحد المقترح لعبارة التهنئة: 120 حرفًا.
- يجب منع النص الطويل من كسر تصميم البطاقة.
- عند الطول الزائد، يتم تقليل حجم الخط أو تقسيم النص على أكثر من سطر.
