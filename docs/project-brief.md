# ملخص المشروع

## اسم مؤقت

بطاقة عيدك

## وصف مختصر

تطبيق ويب خفيف كهدية للمتابعين، يمكّن الزائر من إنشاء بطاقة تهنئة عيد باسمه أو باسم شخص عزيز، ثم تحميلها كصورة PNG ومشاركتها عبر واتساب ووسائل التواصل.

## الجمهور المستهدف

- متابعو صاحب الحساب في X أو تيليجرام أو واتساب.
- أشخاص يريدون تهنئة سريعة وأنيقة قابلة للتخصيص.
- مستخدمون يدخلون غالبًا من الجوال.

## القيمة الأساسية

بدل أن يرسل المستخدم تهنئة عامة، يستطيع خلال ثوانٍ إنشاء بطاقة شخصية مثل:

> إهداء إلى عمي الغالي محمد

ثم تحميلها ومشاركتها.
