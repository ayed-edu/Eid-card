# CLAUDE.md — قواعد العمل على هذا المشروع

## قواعد إلزامية

1. **لا تعيد بناء ما يعمل.** إذا كانت الميزة موجودة وتعمل، لا تلمسها.
2. **مهمة واحدة فقط في كل رد.** لا تجمع عدة تعديلات في رد واحد.
3. **لا تكتب الكود كاملاً في الرد.** استخدم Edit وليس Write إلا عند إنشاء ملف جديد.
4. **حدّث docs/status.md بعد إنجاز كل مهمة.**
5. **لا تغيّر أسماء الملفات أو تنقل الكود بين الملفات.**
6. **لا تعمل refactor** إلا إذا طُلب صراحةً.
7. **راجع docs/remaining-tasks.md** قبل بدء أي مهمة لتعرف الأولوية الصحيحة.
8. **راجع docs/known-issues.md** قبل لمس أي منطقة حساسة.
