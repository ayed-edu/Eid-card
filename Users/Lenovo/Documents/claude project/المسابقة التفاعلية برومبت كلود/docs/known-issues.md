# المشاكل والمخاطر المعروفة

## مخاطر محتملة

### ISSUE-01 — PDF + عربي + RTL
- **الخطورة**: متوسطة
- **التفاصيل**: مكتبات PDF الافتراضية (print/html2canvas) لا تدعم RTL بشكل كامل
- **المنطقة**: `screen-results.js` — دالة التصدير
- **التوصية**: اختبر على متصفح Chrome فعلياً قبل إغلاق هذه المهمة

### ISSUE-02 — Web Audio API على iOS/Safari
- **الخطورة**: متوسطة
- **التفاصيل**: iOS يمنع تشغيل الصوت قبل تفاعل المستخدم الأول
- **المنطقة**: `audio-engine.js`
- **التوصية**: التأكد أن AudioContext يُنشأ بعد click/touch وليس عند تحميل الصفحة

### ISSUE-03 — حجم CSS (4286 سطر في ملف واحد)
- **الخطورة**: منخفضة
- **التفاصيل**: صعوبة في العثور على أجزاء محددة
- **التوصية**: لا تعمل refactor — فقط تذكر هذا عند البحث عن أنماط CSS

### ISSUE-04 — localStorage وحجم البيانات
- **الخطورة**: منخفضة
- **التفاصيل**: localStorage محدود بـ ~5MB — إذا تراكمت الأسئلة والجلسات قد يمتلئ
- **المنطقة**: `session-persistence.js`, `app-config.js`
- **التوصية**: تأكد أن الجلسات القديمة تُحذف (max age 24 ساعة موجود، تحقق أنه يعمل)

### ISSUE-05 — الأسئلة النموذجية للرياضيات
- **الخطورة**: منخفضة (محتوى وليس تقني)
- **التفاصيل**: أسئلة الرياضيات النموذجية قد تكون بسيطة جداً أو غير ملائمة للصف الثالث
- **المنطقة**: `data-units.js`
- **التوصية**: مراجعة المحتوى مع المعلم قبل الإطلاق الفعلي

### ISSUE-06 — اختصار لوحة المفاتيح يتعارض مع المتصفح
- **الخطورة**: منخفضة
- **التفاصيل**: بعض الاختصارات (Space, Enter) قد تتعارض مع سلوك المتصفح في حالات معينة
- **المنطقة**: `keyboard-shortcuts.js`
- **التوصية**: تأكد من `preventDefault()` في الأماكن الصحيحة

## ملاحظات تقنية

- الأنيميشن يستخدم Anime.js v4 — تحقق من التوافق مع API الجديدة إذا وجدت مشاكل
- `window.appState` هو الحالة العامة — احذر من التعديل المتزامن على نفس الخاصية
- لا يوجد نظام error handling مركزي — الأخطاء قد تُصمت silently
