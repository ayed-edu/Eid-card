# تطبيق بطاقة تهنئة العيد

تطبيق ويب بسيط يعمل من GitHub Pages، يسمح للزائر باختيار قالب تهنئة، تعديل اسم المهدى إليه والعبارة، معاينة البطاقة، ثم تحميلها كصورة PNG أو JPG ومشاركتها عبر واتساب أو وسائل التواصل.

## الهدف

تقديم هدية رقمية خفيفة للمتابعين في العيد، قابلة للانتشار، وتُظهر هوية صاحب المشروع بشكل لطيف وغير مزعج.

## النطاق الحالي

- تطبيق ويب ثابت يعمل بدون Backend.
- الاستضافة عبر GitHub Pages.
- 3 قوالب تهنئة مبدئية.
- تعديل النصوص مباشرة.
- معاينة مباشرة للبطاقة.
- تصدير PNG كخيار أساسي.
- JPG خيار ثانوي لاحق.
- مشاركة نص مخصص يحتوي اسم المهدى إليه.
- تحميل الصورة كخيار مضمون.

## التشغيل المتوقع

يفتح المستخدم الرابط، يختار التصميم، يكتب اسم المهدى إليه والرسالة، يعاين البطاقة، ثم يحمّل الصورة أو يشاركها.
