# قواعد العمل للوكيل

- لا تبنِ المشروع من الصفر بعد بدء التنفيذ إلا بطلب صريح.
- نفّذ مهمة واحدة صغيرة في كل مرة.
- لا تضف Backend أو قاعدة بيانات.
- التطبيق يجب أن يعمل على GitHub Pages.
- اجعل المخرج الأساسي PNG.
- حافظ على بساطة الكود وسهولة تعديله.
- لا تستخدم مكتبات ثقيلة إلا عند الحاجة.
- لا تعتمد على صور أو خطوط خارجية بشكل يمنع التصدير الصحيح.
- كل النصوص العربية يجب أن تظهر باتجاه RTL وبمحاذاة مناسبة.
- قبل أي تعديل كبير، حدّث docs/status.md و docs/tasks.md.
